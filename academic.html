Real World Project
Felucca, sponsered by CMU-CERT
Github Repo

I'm one of the main members in Felucca Development Team.
The CERT Executable Code Analysis team develops and maintains a set of program analysis tools, known as Pharos that are based on an increasingly complex infrastructure. The purpose of Felucca project is to develop a control center to unify the Pharos tools and make it easier to manage. Felucca, as a control center, would be able to assist program analysts or users from academic field to run, monitor, and export results from Pharos tools using a single user interface. It also allows The CERT to update the new version of Pharos tools set without any modification in Felucca control center.


Course Related Project

Distributed Syetem

The following are the four projects that I finished in courese 15-640.

Transparent Remote File Operations Project. I build an RPC system to allow remote file operations (open, close, read, write, lseek, stat, unlink, getdirentries, non-standard call). In detail. I create a server process to provide the remote file services AND create a client stub library to perform RPCs.

File-Caching Proxy Project. I design and implement a server that lets a remote client read, modify, and delete files in local file space with no security provisions.
In detail, I created a proxy that sits between the client and the server to handle the file operation RPCs from one or more concurrent clients, fetch needed files from the server, and push modifications back to the server.
I also created a server that stores and provides access to a set of files, operate at the granularity of files, and handle concurrent connections from multiple proxies.

Implementation and Performance Tuning of a Scalable Web Service. In this project, I implemented various techniques to scale out a simulated, Cloud-hosted, multi-tier web service (a web store front). Then I evaluated the system to understand what is the bottleneck at different loads, and use this information to decide when to scale out which components in order to improve performance.

Two-phase Commit for Group Photo Collage Project. In this project, I built a part of a system that generates and publishes group collages assembled from multiple images contributed by multiple nodes concurrently. The system can keep track of the the status of each node for the future recovery of accidental failure, so each transaction can be resolved as if there was no failure.

Link Analysis. In this project, I developed PageRank algorithm with different personalized variants (PageRank, Personalized PageRank, and Query-sensitive PageRank) and compare them empirically on CiteEval dataset

Deep Learning for Text Classification. In this assignment,I developed SGD solver and Newton solver, two optimization methods to solve the objective function of SVM. In addition, I implement SGD optimization on the PMF method that was introduced in the earlier Collaborative Filtering Assignment.

Physics Research

The following are the projects that I woked on when I was an undergraduate major in Physics.

Design of Multi-channel CNN to Predict 3D Motion of Galaxy

Prof. Kenji Bekki (International Centre for Radio Astronomy Research, UWA)         Sep 2018 - Aug 2019

We designed a multi-channel CNN structure for hydrodynamical simulation images prediction. After training the CNN using different sets of simulated images with different astronomical parameters and smoothing the real observational images, this CNN has ability to get accurate predicted parameters of 3D motions of the galaxy.

Prediction of Properties of New 2D material with DFT and Machine Learning

Advisor: Prof. Wenguang ZHU (School of Physical Science, USTC)               May 2019 - Aug 2019

We applied data of different material from Database to density functional theory calculations (DFT) to get the band structures, extracted the features from band structures of different 2D material, and marked them as label in machine learning training. Then, we developed a supervised machine learning algorithm based on the above results to design new 2D Ferroelectric and Piezoelectric materials with optimal performance.
